# Comparative Study of Federated Learning Frameworks

Repositorio asociado al Trabajo Fin de Grado de Raúl Luján Domenech.

El proyecto compara tres frameworks de aprendizaje federado aplicados a clasificación de imágenes:

- Flower
- NVIDIA FLARE
- FedML

## Estructura

```text
src/
  flower/     Código final de los experimentos con Flower
  nvflare/    Código final de los experimentos con NVIDIA FLARE
  fedml/      Código final de los experimentos con FedML

memoria/
  main.pdf    Memoria final del TFG
  main.tex    Documento principal LaTeX
  contenido/  Capítulos de la memoria
  referencias.bib

resultados/
  README.md   Nota sobre los resultados finales
```

## Nota

No se incluyen modelos entrenados, workspaces internos, cachés, logs completos ni archivos auxiliares de compilación.

La memoria final y el código utilizado para el estudio quedan recogidos en este repositorio.
